This application note demonstrates interfacing AT89LP microcontrollers
to the AT93CXX 3-wire type of serial EEPROM.  The text for "Interfacing
AT93CXX Serial EEPROMs with AT89LP Microcontrollers" can be found on the 
Atmel Web site www.atmel.com.

The following files pertain to the application note:

AT93CXX.SCH        ORCAD schematic file
AT93CXX.ASM        80C51 assembly source code
SERIAL.LIB         ORCAD library file for the schematics
